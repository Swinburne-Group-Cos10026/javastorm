Java Developer

Job Overview

The position is anchored in JAVASTORM Digital Product Department, responsible for delivering the best value in software applications for our internal and external clients. As a Java Developer, you will be responsible for designing, developing, testing, and maintaining Java-based applications according to the agreed enterprise architecture. You will work closely with a team of developers and project managers to deliver high-quality software as solutions to business problems in partnership with external Product owners. You will also be responsible for troubleshooting and debugging issues, as well as staying up-to-date with emerging trends and technologies in Java development.

With most large organizations using Java to implement software systems and backend services, a Java developer is one of the most sought-after jobs today.

The Java Developer reports to Development Team Leader and serves in the specialist team.

Key Responsibilities

  1. Working with business and technology stakeholders to understand and analyse requirements for new features and how they will impact the existing application.
  2. Develop flowcharts, layouts and documentation to identify requirements and solutions.
  3. Design and implement Java applications with well-written code that runs efficiently and optimally.
  4. Contribute to software verifications plans and quality assurance procedures.
  5. Comply with project plans, industry standards and regulatory requirements.
  6. Knowledge sharing and conduct code reviews of other team members.
  7. Stay up-to-date with emerging trends and technologies in Java development.

Benefits

  1. A competitive salary with annual performance-based bonus and stock option schemes.
  2. A pension plan; an employer funded health and medical plan; life insurance plan; long term disability coverage; paid time off; an employee assistance program; and a discount platform.
  3. Provide a technology stipend or allowance to cover the cost of purchasing or upgrading equipment and software.

Skills and Qualifications

Essential

  • Advanced skills in Java 11 or later.
  • Experience with popular Java frameworks such as Spring, Hibernate, and Struts.
  • Familiarity with software development tools such as Eclipse, IntelliJ IDEA, and Git.
  • Sound knowledge of Object-Oriented Programming (OOP) Patterns and Concepts
  • Basic knowledge of Web Technologies like HTML, JavaScript, CSS, JQuery
  • Excellent problem-solving and analytical skills.
  • Strong communication and collaboration skills.
  • High sense of initiative to work without a close supervision.

Preferable

  • Familiarity with Agile development methodologies.
  • Bachelor's Degree or Master's degree in Computer Science, Information Technology, or a related field.
  • Certification in Java programming (at least 3 years).
  • Experience with cloud computing platforms such as Amazon Web Services (AWS) or Microsoft Azure.
  • Basic Understanding of the concepts of MVC (Model-View-Controller) Pattern, JDBC (Java Database Connectivity).
