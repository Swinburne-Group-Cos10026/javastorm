Security Analyst

Job Overview

This role is part of the JavaStorm Digital Security Department and specializes in software solution security initiatives across the group. In other words, it is the focal point for all security-related concerns from different aspects, including the network and data of the software application. The incumbent will have to maintain the confidentiality, integrity, and availability of the technology infrastructure in the developed software. The role extends to identifying and mitigating potential security risks and vulnerabilities in the software solution before any breaches occur. This role closely collaborates with the developer team, quality assurance team, and project managers to ensure the security of the solutions while ensuring they comply with industry standards and regulations. It is also accountable for conducting security assessments and providing recommendations to improve the system's security posture.

The Security Analyst reports to the Project Manager throughout the software development lifecycle.

Key Responsibilities

  1. Conduct security assessments of the software applications and infrastructure to identify and address potential security risks and vulnerabilities.
  2. Perform triage, in-depth investigation, and remediation to all detected security incidents and breaches in a timely fashion.
  3. Ensure the software solutions are secured from unauthorized access or activity while maintaining the relevant data.
  4. Capture and document security incidents, findings, and responses in an organized manner.
  5. Work with developers and project managers to implement security best practices throughout the software development processes.
  6. Stay up-to-date with emerging trends and technologies in information security and incorporate those into the software applications.
  7. Ensure the software solutions comply with industry standards and regulations, such as PCI-DSS, HIPAA, and GDPR.

Skills and Qualifications

Essential

  • Bachelor's degree in Computer Science, Information Technology, Computer Engineering or equivalent.
  • At least three years of experience in information security, preferably in a software development environment.
  • Intermediate coding skill in Python, PowerShell, or Jenkins is required.
  • Strong understanding of information security concepts and practices, such as cryptography, network security, and application security.
  • Experience with security testing tools and techniques, such as vulnerability scanning and penetration testing.
  • Familiarity with software development tools such as Git and Jenkins.
  • High-energy work ethic, with the ability to establish a vision, drive change, and deliver results.
  • Good anticipation, analytical, and problem-solving skills.
  • Excellent verbal and written communication skills, especially in English, interpersonal, and teaching skills.

Preferable

  • Master's Degree in Information Security or Cybersecurity.
  • At least two years of experience with threat-hunting, analysis, and triage of cyber security threats, including exposure to incident response, vulnerability management, and attack testing and validation.
  • CISSP, CISA, CISM, or other industry-recognized security certification(s).
  • Basic knowledge of cloud computing platforms such as Amazon Web Services (AWS) or Microsoft Azure.
  • Familiarity with Agile development methodologies.
